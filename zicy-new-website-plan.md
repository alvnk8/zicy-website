# Zicy — New Website Plan & Information Architecture

**Goal:** Reposition zicy.com from a self-serve "AI butler" into the **truth layer for AI visibility** — a closed-loop platform (measure → diagnose → act → prove) with dedicated landing pages for all four ICPs.
**Screenshots:** Placeholders below reference the dashboard screenshots already on record. They are temporary; the dev/design team will regenerate clean versions.

---

## A. What changes from the current site (and why)

| Current | New | Why |
|---|---|---|
| "AI butler," "affordable," "no agency required" | "The truth layer for AI visibility" | The butler/SMB voice repels enterprise brands, PR firms, agencies, publishers |
| One generic homepage | Homepage + 4 ICP solution pages | Each ICP buys for a different reason; one-size messaging converts none of them well |
| Flywheel: Diagnose→Prioritize→Execute→Measure | Loop: **Measure → Diagnose → Act → Prove** | Same bones, sharper: leads with truth/measurement, ends with proof of real demand |
| Feature lists ("Four Pillars") | Module proof tied to ICP outcomes | Features don't sell; outcomes per ICP do |
| Self-serve only | Self-serve **+** done-for-you path | Captures both the scalable SaaS motion and the enterprise/managed (Growth.pro) demand |

**Voice:** keep the plain-language accessibility, but add enterprise credibility and Alvin's house style — outcome-first, evidence-backed, loss-aversion framing. Every page ladders back to the spine: *competitors measure; Zicy measures, fixes, and proves — across every engine and every language.*

---

## A2. Growth.pro IP — reference, don't claim (locked)

Growth.pro's three-layer IP — the **AI Visibility Operating System** (the offer), the **PAVA Framework** (Presence · Authority · Visibility · Amplification — the methodology), and **AI Citation Enablement / AICE** (the practice) — stays **Growth.pro's**. Zicy **references** it; Zicy does not adopt it as its own. (Per PK v3.6: these are Growth.pro's consultancy IP.)

- **Zicy keeps its own positioning** — the truth layer + the measure → diagnose → act → prove loop. Zicy is NOT rebranded as "the AI Visibility Operating System," and does NOT adopt Growth.pro's "Earn the Answer." tagline.
- **Reference appears in three places only:** (a) a light provenance line — *"Built by Growth.pro, the consultancy behind the PAVA framework for AI visibility"*; (b) the **About** page (the Growth.pro / Zicy relationship); (c) the **Done-for-you** page, where the frameworks are the Growth.pro *service* differentiator.
- **Always attributed to Growth.pro** — "Growth.pro's PAVA framework," "Growth.pro's AI Visibility Operating System." Never unattributed, never positioned as Zicy's own product framework.
- **Policy-clean carry-over (from PK v3.6):** wherever the IP is referenced, no "engineering" or manipulation vocabulary — use "Enablement," "earn," and brand-side framing. The IP only holds as a moat on the endorsed side of Google's policy line.
